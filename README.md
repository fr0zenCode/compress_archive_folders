## Установка и запуск
____
### 1. Установить Python версии 3.12+ с [официального сайта](https://www.python.org/)
### 2. Установить Poetry, используя *pip*
```bash
pip install poetry
```
### 3. Установить зависимости
```bash
poetry install 
```
### 4. Запустить программу
```bash
python main.py 
```
